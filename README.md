## Requisitos

- git
- docker


## Instalação

Use o git para baixar o projeto:

```bash

    $ git clone https://github.com/jmreis/jogo-da-velha.git


```

## Usando Docker

Depois de baixar execute os comandos no terminal:

```bash

    $ cd clone-netflix

    $ docker build -t jogo-da-velha .

    $ docker run -d -p 8000:80 jogo-da-velha
 

```
